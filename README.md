# Book Library Collection

A React app showcasing a collection of books using reusable components, built with Vite.

## Components
- **BookCard**: Reusable component to display individual book information (title, author, genre, year, description).

## Links
- **GitHub Repository:** https://github.com/<yourusername>/book-library
- **Live Website:** https://<your-deployment-url>

## Run Locally
1. Clone the repository  
   `git clone https://github.com/<yourusername>/book-library.git`
2. `cd book-library`
3. `npm install`
4. `npm run dev`

## Deployment
App was deployed using Vercel (or Netlify). Build command: `npm run build`. Publish directory: `dist`.

## Project Structure
- `src/components/BookCard.jsx`
- `src/App.jsx`
- `src/index.css`

## Notes
- Each `BookCard` item uses a unique `key` prop.
